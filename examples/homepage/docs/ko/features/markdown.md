# 마크다운 확장

Simpesys는 확장된 기능을 위해 다양한 마크다운 문법과 기능을 제공한다. 이 문서는 Simpesys가 지원하는 마크다운 기능과 문법 확장을 설명한다.

## 플러그인

### 각주

각주 문법을 지원한다. 각주는 백링크와 함께 문서 끝에 렌더링된다.

```markdown
여기에 인용이 필요한 문장이 있다[^1].

[^1]: 각주 내용이다.
```

### 수식 표기

수식은 KaTeX로 렌더링된다. 달러 기호 구분자를 사용한다.

| 문법 | 설명 |
|------|------|
| `$...$` | 인라인 수식 |
| `$$...$$` | 디스플레이 (블록) 수식 |

### 이모지

이모지 단축 코드는 유니코드 이모지로 변환된다.

```markdown
:smile: :rocket: :thumbsup:
```

### 작업 목록

체크박스 문법을 지원한다. 체크박스는 비활성화된 (읽기 전용) 입력으로 렌더링된다.

```markdown
- [x] 완료된 작업
- [ ] 대기 중인 작업
```

### 이미지 크기 조정

이미지 크기를 지정할 수 있다.

```markdown
![대체 텍스트](image.png =200x100)
![대체 텍스트](image.png =200x)
![대체 텍스트](image.png =x100)
```

### 외부 링크

외부 도메인에 대한 링크에는 `external` 클래스가 설정된다. 내부 도메인은 `config.web.domain` 옵션에 의해 결정된다.

### 컨테이너 블록

커스텀 컨테이너 블록은 콜아웃 스타일 포맷을 제공한다. 컨테이너 블록은 CSS 스타일링을 위한 해당 클래스 이름과 함께 `<div>` 요소로 렌더링된다.

#### 토글

```markdown
::: TOGGLE 요약 텍스트
확장하면 표시되는 숨겨진 내용.
:::
```

#### 노트 블록

```markdown
::: NOTE
노트 콜아웃이다.
:::
```

#### 정보 블록

```markdown
::: INFO
정보 콜아웃이다.
:::
```

## HTML 허용

마크다운 콘텐츠에서 raw HTML을 허용한다. HTML은 수정 없이 출력으로 통과된다.

```markdown
<div class="custom-wrapper">
  <p>커스텀 HTML 콘텐츠</p>
</div>
```
