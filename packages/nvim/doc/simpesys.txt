SIMPESYS                                                          *simpesys*

NeoVim plugin for the Simpesys digital garden system. Provides link
navigation, autocompletion, and diagnostics via an LSP server backed
by @simpesys/core.

Requires NeoVim 0.11+. The LSP server is registered via
`vim.lsp.config()` and `vim.lsp.enable()`, so standard LSP keymaps
(e.g. `gd`, `gr`, `K`) are handled by your global NeoVim configuration.

==============================================================================
SETUP                                                       *simpesys-setup*

Requires `deno` in PATH. The plugin auto-detects Simpesys projects by
looking for `simpesys.metadata.json` or `deno.json` referencing
`@simpesys/core` in parent directories.

>lua
  require("simpesys").setup({
    -- Optional: override LSP server command
    server_cmd = nil,
  })
<

==============================================================================
LINK STYLES                                           *simpesys-link-styles*

  simpesys:   `[[key]]` or `[[key]]{label}`
  obsidian:   `[[key]]` or `[[key|label]]`

Configured via `docs.linkStyle` in project config.
